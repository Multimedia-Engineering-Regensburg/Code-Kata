Code Kata

Implementieren Sie die Lösungen für diese Aufgaben in der app.js-Datei des jeweiligen Unterordners. Die passendne Javascript-Anwendung können Sie über diese Links erreichen. Die Aufgabenbeschreibung sehen Sie direkt in der jeweiligen Anwendung (oben rechts, hinter dem Fragezeichen).

  1. Color Sequence (Click-Events speichern und zeitgesteuerte CSS-Manipulation)
  2. Parser (Einzelne Teile eines Textes durch zusätzliche Element und CSS-Klassen gestalten)
  3. Image Transfer (Inhalte eines Image-Container auf einen Canvas kopieren und filtern)
  4. Image Zoom (Dynamsiche Zeichen eines Bildausschnitts)
